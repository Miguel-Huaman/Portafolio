# Tus primeros pasos con 'React.js'

## Librería para iniciar proyecto
`npx create-react-app nombre-de-carpeta`

## Instalación de yarn
`npm install -g yarn`

## Subir a producción
`yarn build`

## Creación proyecto mediante Yarn
`npx create-react-app Nombre_de_proyecto`
